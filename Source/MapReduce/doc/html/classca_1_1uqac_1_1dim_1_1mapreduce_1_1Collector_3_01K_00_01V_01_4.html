Detailed Description

Data source used both as the input and output of the map and reduce phases.

A Collector can be used to:

  1. Store data tuples using the collect method
  2. Enumerate data tuples using the hasNext and next methods, like an Iterator
  3. Partition the set of tuples into a set of Collectors, grouping tuples by their key, using the subCollector and subCollectors methods
